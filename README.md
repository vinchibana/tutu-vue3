# Erabbit

该项目实现了电商平台主线业务功能，电商首页、一级分类、二级分类、商品详情、购物车、结算、支付、个人中心、订单管理、订单详情。项目采用前后端分离模式，前台使用了 Vue3.0技术栈构建，并封装自有 UI 组件库，接入 QQ 登录，支付宝支付业务，以及前端 SKU 解决方案，本地线上购物车解决方案。
